# AI Insights API

Access AI-generated analysis and strategic insights.

## Get My Insights
**GET** `/ai-insights/me`

Retrieves insights relevant to the logged-in user's active cases.

---

## Dashboard Summary
**GET** `/ai-insights/dashboard/summary`

Provides a high-level summary of risks and opportunities across all cases.

---

## Recent Insights
**GET** `/ai-insights/recent`
